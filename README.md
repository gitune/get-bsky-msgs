# get-bsky-msgs
Get my Bluesky posts in python

## これは何？
AT Protocol SDK(Python)を使ってBlueskyから自分のポストを取得するscriptです。なお後で使うためにtext中のURLの復元と添付されたイメージのfullsize URLの取得も行っています。
